# Laravel App

## Start Laravel App

```bash
$ cd examples/laravel-auth-mysql
$ bolt up
```

## Stop Laravel App

```bash
$ cd examples/laravel-auth-mysql
$ bolt down
```

## Log Laravel App

```bash
$ cd examples/laravel-auth-mysql
# In watch mode
$ bolt log --follow laravel
# Onetime logging
$ bolt log laravel
```

# Expo & Express APIs

## Start

```bash
$ cd examples/expo-express-apis
$ bolt up
```

## Stop

```bash
$ cd examples/expo-express-apis
$ bolt down
```

## Log Expo App

```bash
$ cd examples/expo-express-apis
# In watch mode
$ bolt log --follow expo
# Onetime logging
$ bolt log expo
```

## Log Express APIs

```bash
$ cd examples/expo-express-apis
# In watch mode
$ bolt log --follow apis
# Onetime logging
$ bolt log apis
```
